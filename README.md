---

## 三、LINE OA 經營（加好友推播）

除了點餐系統，「加好友變好友、方便發廣告訊息」就是 **LINE Official Account** 本身的行銷功能：

- 任何上述點餐系統都會綁定 LINE OA
- 好友數越多，就能用 LINE OA 後台發 **推播訊息**（免費帳戶每月 500 則，付費可無限推播）
- 也可以搭配聊天機器人、點餐優惠引導加好友

---

## 簡要建議

- **如果只要便宜先有 LINE 點餐** → SWD Order（$399/月起），之後有需要再接 POS
- **如果咖啡廳剛開幕想一次到位** → iCHEF 或 料理人 BeeSmart，台灣餐飲最常用、售後資源多
- **如果預算有限但想完整功能** → TAKOPOS 或 你訂 Nidin，CP 值不錯
